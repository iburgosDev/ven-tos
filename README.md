# Ven-tos 

Plataforma donde personas ofrecen sus talentos y otros reservan su tiempo.

## 🌐 Demo en producción

- **Frontend:** https://ven-tos.vercel.app
- **Backend API:** https://ven-tos.onrender.com/docs

## 🛠️ Stack

- Python + FastAPI + SQLAlchemy
- React + Vite
- PostgreSQL + Redis
- Docker + Render + Vercel

## 👤 Roles

- **Talento:** publica servicios, gestiona agenda, ve métricas
- **Cliente:** explora servicios y hace reservas

## 🗺️ Fases

| Fase | Contenido | Estado |
|------|-----------|--------|
| 1 | Docker, PostgreSQL, Redis, modelos, auth JWT | ✅ |
